# 一. 实现三个数从小到大排序并输出
任意输入三个数，编程实现对这三个整数由小到大输出。 
## 运行结果
#### 例1
```
5 3 9
3 5 9
```
#### 例2
```
 1 2 3
 1 2 3
```

# 二. 猴子吃桃问题
猴子第一天摘下若干个桃子，当即吃了一半，还不过瘾，又多吃了一个。第二天早上又将第一天剩下的桃子吃掉一半，有多吃了一个。以后每天早上都吃了前一天剩下的一半零一个。到第 10 天早上想再吃时，发现只剩下一个桃子了。编写程序求猴子第一天摘了多少个桃子。
## 运行结果
```
1534
```
